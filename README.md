# App Beat

Aplicação web construída com React, TypeScript e Vite.

## Pré-requisitos

- Node.js 20+
- npm 10+

## Fluxos principais

### Desenvolvimento web

```bash
npm install
npm run dev
```

### Build web (produção)

```bash
npm run build
```

## Scripts úteis

- `npm run lint`: executa o ESLint.
- `npm run preview`: sobe uma versão estática da build.
- `npm run storybook`: desenvolve componentes isolados com Storybook.
